
#### 如何使用？
##### 安装
```
npm install egrow-vue-generator --save
```
##### 使用配置文件
1.  generator.config.js为配置项文件
2.  generator.data.js为数据来源文件
##### 在scripts中添加命令
```
"generate": "generate"
```
##### 启动
```
npm generate
```
#### 发布版本
```
npm publish
```
#### 启动命令
```
node index.js
```
#### 数据模型文件
```
model.js
```
